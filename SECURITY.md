# Security Policy

## Supported Versions
Only the latest version on the `master` branch is currently receiving security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Because this tool handles sensitive (though local/offline) medical data, security is paramount.
If you discover a vulnerability, especially related to the AES encryption or PBKDF2 key derivation, **please do not open a public issue.**

Instead, please email the project maintainer directly. We will strive to acknowledge your report within 48 hours and release a patch as soon as possible.
